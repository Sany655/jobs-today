# jobs today

php laravel project
[visit live link](http://mazharsany.xyz/jobs-today)

## setup instruction

* clone the project
* composer install
* cp .env.example .env => if .env file not exists
* php artisan key:generate
* npm install => if package.json has any dependency
* create a database like .env file
* php artisan migrate
* php artisan db:seed
